AI coding agents are powerful — but they can make catastrophic mistakes. A misunderstood instruction, an over-eager cleanup step, or a poorly scoped refactor can result in commands like `rm -rf ~/`, `git reset --hard`, or `git checkout -- .` wiping out work you can't get back. Soft rules written in `CLAUDE.md` or `AGENTS.md` help set expectations, but they rely on the model following instructions — and models don't always follow instructions. CC Safety Net is a hard technical constraint: a PreToolUse hook that intercepts every shell command your AI agent attempts to run and blocks destructive operations **before they execute**, using semantic command analysis rather than fragile pattern matching. It works across Claude Code, Codex, Gemini CLI, GitHub Copilot CLI, Kimi Code, OpenCode, and Pi, so your code, history, and filesystem stay intact no matter which agent you're working with.
